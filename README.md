Chat App for Sailer AI Front End Developer Position.

## Getting Started

First, install the packages:

```bash
npm i
# or
yarn
```

Then, start the server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser .


## .env Example

```bash
API_URL=http://localhost:8000
```
